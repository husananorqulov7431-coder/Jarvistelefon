# Jarvis Phone Assistant

Android assistant with:
- app launching
- web search
- SMS compose/send flow
- dialer flow
- clipboard
- TTS and speech recognition
- multi-provider AI failover via OpenAI-compatible endpoints

## Setup
Edit `app/src/main/assets/providers.json` and paste your API keys in `apiKey`.

## Build
Open in Android Studio, sync Gradle, and run on a device.
